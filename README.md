# Water-Ring-Toss

Water Ring Toss – Childhood Recreated

An online recreation of the classic Water Ring Toss toy that many of us grew up playing.

Built using HTML, CSS, and JavaScript, this project simulates the nostalgic handheld water game where you press buttons to push floating rings onto pegs.

# The Story Behind This Project

When I was a child, I had a small water ring toss toy that I absolutely loved.
It was simple — just water, colorful rings, and tiny pegs — but it made me so happy.

As I grew up, I couldn’t find that toy anywhere anymore.
I searched for it, but it felt like a small piece of my childhood had disappeared.

So instead of just missing it…
I decided to build it myself.

This project is not just a game —
it’s my way of recreating a childhood memory using code.

What once existed in plastic and water
now lives in HTML, CSS, and JavaScript.

# Features

Interactive left and right controls

Floating ring movement simulation

Score counter

Reset functionality

Simple, clean, colorful UI inspired by the original toy
